We use a rolling average to smooth out the noise - this sets the window size for this average To calculate the maximum value we use a larger window Finally we set the tolerance of how close a value has to be to this maximum to be the maximum

#rolling window size - for the rolling average of water data
wsz<-3
#maximum window size - for the number of points from which to calculate the maximum
msz<-3
#how close to the maximum value does a value have to be to be counted as a maximum
csz<-0.95
